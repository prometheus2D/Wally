# RequirementsExtractor — Actor Docs

This folder contains actor-private reference material for the RequirementsExtractor.

Place any of the following here to give the actor additional context:
- Domain glossaries
- Existing requirements templates or standards the team follows
- Stakeholder interview guides or question frameworks
- Example requirements documents for the project

Files in this folder are listed in every prompt under `## Documentation Context`
so the LLM can consult them without being explicitly told to.
