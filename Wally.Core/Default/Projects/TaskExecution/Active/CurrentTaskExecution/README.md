# Current Task Execution

This folder is the scratch space for `ExecuteTasksLoop`.

- `ExecutionState.md` is the routed-loop resume state.
- `TrackerState.json` stores the selected tracker path and current task number.
- `SelectedTask.md`, `ExecutionResult.md`, `VerificationResult.md`, and `Outcome.md` capture the current one-task execution flow.

The loop may reset this folder on a new run while preserving this README.
